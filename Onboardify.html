Project Overview

Onboardify is a smart employee onboarding platform developed as part of an academic project. It combines a JavaFX desktop application and a Symfony-based web application to deliver a structured, engaging, and interactive onboarding experience. The platform aims to enhance new hire integration through resource management, interactive modules, and employee well-being support — while ensuring real-time visibility for HR and managers. Key features include:

  • Secure login and sign-up system
  • Project assignment and tracking module
  • Interactive quizzes for training validation
  • Posts and reclamation system for feedback and communication
  • Well-being program integration to support employee mental health

Tech Stack: JavaFX · Symfony · PHP · MySQL · HTML/CSS · JavaScript
